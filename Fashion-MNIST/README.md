## Details
<b>Model:</b> Multi-layer Perceptron
<br>
<b>Architecture:</b> 6 fully-connected layers with dropout as below
<br>
<i>784 (input) -> 512 (hidden1) -> 512 (hidden2) -> 256 (hidden3) -> 128 (hidden4) -> 64 (hidden5) -> 10 (output)</i>
